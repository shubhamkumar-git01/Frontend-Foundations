# 05 Mini Foundation Projects 🏗️

Knowledge is useless without application. After mastering HTML, CSS, and Vanilla JS, I built these mini-projects entirely from scratch. No libraries, no frameworks. 

## Projects Included:

### 1. Weather Dashboard
- **Goal**: Fetch real-time data from a public API and dynamically update the DOM.
- **Concepts used**: Fetch API, Async/Await, JSON parsing, DOM manipulation.

### 2. Expense Tracker
- **Goal**: Manage application state and persist data.
- **Concepts used**: High Order Functions (reduce/filter), LocalStorage, HTML forms, event delegation.

> *Building these made me realize why frameworks like React were invented. Managing state purely via DOM manipulation gets extremely messy! But I am glad I did it the hard way first.*
