# Frontend Foundations

A comprehensive collection of practice code, notes, and mini-projects built while mastering the core pillars of web development: **Semantic HTML5, CSS3 Architecture, and Vanilla JavaScript**.

This repository serves as a personal reference guide and a solid foundation before moving on to modern frameworks like React and Next.js.

---

## 📂 Repository Structure

The practice files are organized into five main modules, each containing specific sub-topics with code examples and detailed notes.

### [00-Web-Basics](./00-Web-Basics)
Core theoretical concepts of how the internet and browsers function.
- DNS and TCP/IP Architecture
- The Critical Rendering Path

### [01-HTML-Deep-Dive](./01-HTML-Deep-Dive)
Semantic structure, forms, and accessibility.
- Basic Structure & Text Formatting
- Lists, Tables, Forms & Inputs
- Semantic Tags & Multimedia
- Links, Iframes, Meta Tags (SEO)
- SVG, Canvas & ARIA Accessibility

### [02-CSS-Architecture](./02-CSS-Architecture)
Styling, layouts, and responsive design principles.
- Selectors, Specificity & The Box Model
- Typography, Colors & Positioning
- Flexbox & CSS Grid (Basics & Advanced)
- Animations, Transitions & Media Queries
- Pseudo-classes, Z-index & Stacking Context
- CSS Variables, BEM Methodology & Reset Architectures

### [03-JS-Core-Concepts](./03-JS-Core-Concepts)
The fundamentals of JavaScript and how the engine processes code.
- Variables, Data Types & Type Coercion
- Operators, Logic, Functions & Scope
- Arrays, Loops, Objects & `this` keyword
- Hoisting, Closures & Prototypal Inheritance
- Error Handling, Math/Date Objects, Maps/Sets & Regex

### [04-JS-Advanced-DOM-Async](./04-JS-Advanced-DOM-Async)
Interacting with the browser and handling asynchronous operations.
- DOM Manipulation & Event Delegation
- Callbacks, Promises & Async/Await (Fetch API)
- ES6 Modules & Web Storage APIs
- Timers, Browser Object Model (BOM) & Web Workers

### [05-Mini-Foundation-Projects](./05-Mini-Foundation-Projects)
Practical applications built with pure Vanilla JavaScript to tie the concepts together.
1. **Color Flipper:** Basic DOM manipulation.
2. **Counter App:** Simple state management.
3. **Expense Tracker:** Advanced array methods and LocalStorage.
4. **Weather Dashboard:** Async API fetching and error handling.
