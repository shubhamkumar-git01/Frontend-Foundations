# Frontend Foundations Masterclass 🚀

This repository represents the very beginning of my 4-year long journey into Full Stack Web Development. I've stripped away the noise and documented the absolute core fundamentals of the web: **HTML, CSS, and Vanilla JavaScript**.

This isn't just a collection of code; it's a comprehensive, deeply practical guide and journal. I've included my raw notes, common pitfalls I faced while learning, and the actual practice code I wrote to solidify these concepts.

## 📂 Repository Structure

1. **[01 HTML Deep Dive](./01-HTML-Deep-Dive)**: Semantic structures, accessibility (a11y), and forms.
2. **[02 CSS Architecture](./02-CSS-Architecture)**: Mastering the Box Model, Flexbox, CSS Grid, and animations.
3. **[03 JS Core Concepts](./03-JS-Core-Concepts)**: Execution contexts, hoisting, closures, and prototypes. The tricky parts of JS.
4. **[04 JS Advanced, DOM & Async](./04-JS-Advanced-DOM-Async)**: The Event Loop, Promises, sync/await, and efficient DOM manipulation.
5. **[05 Mini Foundation Projects](./05-Mini-Foundation-Projects)**: Pure Vanilla JS projects (Weather App & Expense Tracker) to tie everything together.

## 💡 How to use this repository
If you are learning frontend, you can read the README.md inside each folder first for the theoretical notes, and then run the .html and .js files to see the concepts in action. 

> *Note to self: Never skip the basics. The stronger the foundation, the easier React and Next.js will be later.*
